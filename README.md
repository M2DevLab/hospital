## 📌 Features

✨ **Modern UI/UX Design**
A clean and professional healthcare interface designed with user experience in mind.

📱 **Fully Responsive Layout**
Optimized for all screen sizes including mobile, tablet, and desktop devices.

📊 **Animated Counters**
Dynamic statistics (doctors, patients, awards) using Intersection Observer for better performance.

📅 **Appointment Booking Interface**
Interactive form with real-time feedback and confirmation animation.

🧑‍⚕️ **Doctor Filtering System**
Filter doctors by specialty dynamically without page reload.

💳 **Pricing Plans Toggle**
Switch between monthly and yearly plans with smooth UI updates.

🎯 **Smooth Scrolling Navigation**
Enhanced navigation experience with animated scrolling between sections.

⬆️ **Back to Top Button**
Appears after scrolling to improve accessibility and navigation.

📈 **Animated Skill Bars**
Visual representation of hospital performance metrics.

## 🎯 Key Functionalities Explained

### ✔️ Navbar Scroll Effect

The navigation bar dynamically changes style when scrolling, enhancing visibility and UX.

### ✔️ Counter Animation

Counters animate when they enter the viewport using the **Intersection Observer API**, improving performance and avoiding unnecessary rendering.

### ✔️ Doctor Filtering System

Users can filter doctors by specialty using dataset attributes and JavaScript logic.

### ✔️ Pricing Toggle System

Switch between monthly and yearly pricing using a toggle button that updates UI instantly.

### ✔️ Appointment Interaction

The appointment button provides visual confirmation feedback, simulating real-world booking behavior.
